Challenge

The challenge of this lab assignment is to create a function that depends on conditionals while working with a partner.

Problems

There was some issues with having the button appear when testing Michael's code locally. Later we discovered this was due to the ordering of code in the javascript file. The sorting function being on top of the javascript file confused the DOM which led to the code not appearing. This taught us a valuable lesson on how code cascades in Javascript.
